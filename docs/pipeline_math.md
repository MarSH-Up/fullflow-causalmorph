# Pipeline Mathematics

Complete mathematical reference for the full causal learning pipeline.

---

## 1. Problem Statement

Given a multivariate time series $\mathbf{X} \in \mathbb{R}^{T \times p}$ generated by a sequence of $R$ structural causal models (SCMs) with abrupt transitions at unknown times $\tau_1 < \tau_2 < \cdots < \tau_{R-1}$, the pipeline simultaneously:

1. **Detects** the change points $\{\hat\tau_k\}$ from the observed signal.
2. **Recovers** the causal graph $\hat{G}_r$ for each detected regime window.
3. **Aggregates** all estimated regime graphs into a single consensus structure.

Each regime $r$ is governed by a linear SCM:

$$X_i(t) = \sum_{j \in \mathrm{PA}_i^{(r)}} w_{ji}^{(r)} X_j(t) + \varepsilon_i^{(r)}(t), \qquad \varepsilon_i^{(r)} \overset{\text{i.i.d.}}{\sim} \mathcal{L}_i^{(r)}$$

where $\mathrm{PA}_i^{(r)}$ is the parent set of node $i$ in regime $r$, $w_{ji}^{(r)}$ are the edge weights, and $\mathcal{L}_i^{(r)}$ is a non-Gaussian noise distribution (required for LiNGAM identifiability).

---

## 2. Learning Trajectory Generation

### 2.1 Brain-Inspired Structural Evolution

The scenario models a directed learning process: nodes (brain regions) remain fixed, while edges (connections) rewire from an initial sparse configuration $G_\text{init}$ toward a final denser target $G_\text{tgt}$.

**Initial graph** $G_\text{init}$: a random DAG over $p$ nodes generated with connection probability $\rho_0 = 0.6 \cdot \rho_\text{base}$.

**Target graph** $G_\text{tgt}$: a random DAG over the same $p$ nodes generated with connection probability $\rho_1 = 1.4 \cdot \rho_\text{base}$.

Both graphs respect the fixed topological ordering $V_1 \prec V_2 \prec \cdots \prec V_p$, so all edges go from lower to higher indices and any subset forms a valid DAG.

### 2.2 Edge Partitioning

The edge sets are partitioned into three disjoint groups:

$$\mathcal{E}_\text{stable} = \mathcal{E}(G_\text{init}) \cap \mathcal{E}(G_\text{tgt})$$
$$\mathcal{E}_\text{lose} = \mathcal{E}(G_\text{init}) \setminus \mathcal{E}(G_\text{tgt}) \quad \text{(removed over time)}$$
$$\mathcal{E}_\text{gain} = \mathcal{E}(G_\text{tgt}) \setminus \mathcal{E}(G_\text{init}) \quad \text{(added over time)}$$

### 2.3 Scheduled Trajectory

Each edge change is assigned to a specific regime via a uniform schedule. For $k$-th edge in a set of $n$ changes across $R$ regimes ($R-1$ transitions):

$$r^*(k) = 1 + \left\lfloor \frac{k \cdot (R-1)}{n} \right\rfloor, \qquad k = 0, 1, \ldots, n-1$$

This assigns changes as evenly as possible across transitions $1, \ldots, R-1$, with the first change always at transition 1 (guaranteeing $G_0 \neq G_1$).

The edge set for regime $i$ is then:

$$\mathcal{E}(G_i) = \mathcal{E}_\text{stable}
  \;\cup\; \bigl\{ e \in \mathcal{E}_\text{lose} : i < r^*_\text{lose}(e) \bigr\}
  \;\cup\; \bigl\{ e \in \mathcal{E}_\text{gain} : i \geq r^*_\text{gain}(e) \bigr\}$$

By construction: $G_0 = G_\text{init}$, $G_{R-1} = G_\text{tgt}$, and each graph is a valid DAG.

### 2.4 Parametric Variation Within Topology

Each regime $r$ additionally re-samples edge weights and noise scale:

$$\delta_r \sim \mathcal{U}(0.5,\, 1.5), \qquad \sigma_r \sim \mathcal{U}(1.2,\, 1.8)$$

via a deterministic `regime_seed`. Non-stationarity therefore has two sources: topological (edge presence) and parametric (edge weight magnitudes, noise scale).

## 3. Change-Point Detection: Gatekeeper v1-F

### 3.1 Rolling Moment Extraction

For each channel $n$ and each moment $m \in \{1,2,3,4\}$, a rolling window of size $W$ (default 50) is applied:

| $m$ | Statistic | Estimator |
|-----|-----------|-----------|
| 1   | Mean      | $\hat\mu_n(t) = \frac{1}{W}\sum_{s=0}^{W-1} X_n(t-s)$ |
| 2   | Variance  | $\hat\sigma^2_n(t) = \frac{1}{W-1}\sum_s (X_n(t-s)-\hat\mu)^2$ |
| 3   | Skewness  | $\hat\gamma_n(t) = \frac{\hat\mu_3}{\hat\sigma^3}$ where $\hat\mu_k = \frac{1}{W}\sum_s(X_n-\hat\mu)^k$ |
| 4   | Kurtosis  | $\hat\kappa_n(t) = \frac{\hat\mu_4}{\hat\sigma^4} - 3$ |

This yields a moment signal $M_n^{(m)}(t) \in \mathbb{R}^T$ for each channel and each moment.

### 3.2 Continuous Wavelet Transform

Each moment signal is analysed with the Morlet wavelet at $S$ logarithmically-spaced scales $\{s_1, \ldots, s_S\}$ (minimum scale $s_\text{min} = 3.0$, central frequency $\omega_0 = 6$):

$$W_n^{(m)}(t, s) = \int_{-\infty}^{\infty} M_n^{(m)}(u)\; \psi^*\!\left(\frac{u-t}{s}\right) \frac{du}{\sqrt{s}}$$

where the Morlet mother wavelet is:

$$\psi(t) = \pi^{-1/4} e^{i\omega_0 t} e^{-t^2/2}$$

The scalogram (instantaneous power) at each $(t, s)$ is $\mathcal{W}_n^{(m)}(t, s) = |W_n^{(m)}(t, s)|^2$.

### 3.3 Fourier Surrogate Calibration

To establish a null distribution under stationarity, $K = 100$ Fourier surrogates are generated from the baseline segment $\mathbf{X}[\mathcal{B}]$ (first 50% of the first regime):

$$\tilde{X}_n^{(k)} = \mathcal{F}^{-1}\!\left[ |\hat{X}_n(\omega)| \cdot e^{i\phi_\omega^{(k)}} \right], \qquad \phi_\omega^{(k)} \overset{\text{i.i.d.}}{\sim} \mathcal{U}(0, 2\pi)$$

This preserves the power spectrum (autocorrelation structure) while destroying non-stationarity.

**Max-statistic threshold** (FWER control at level $\alpha = 0.40$): for each surrogate $k$ and scale $s$,

$$M_{n,s}^{(k)} = \max_t\; \mathcal{W}_n^{(m)}(t, s; \tilde{X}^{(k)})$$

The per-scale threshold is the $(1-\alpha)$ quantile of the surrogate maxima:

$$\theta_{n,s}^{(m)} = \mathrm{Quantile}_{1-\alpha}\!\left\{M_{n,s}^{(k)}\right\}_{k=1}^{K}$$

Under the null, $P\!\left(\exists\, t : \mathcal{W}_n^{(m)}(t,s) > \theta_{n,s}^{(m)}\right) \leq \alpha$.

### 3.4 Relative Exceedance and K-of-Scales Gate

The normalised exceedance above threshold is:

$$Z_n^{(m)}(t, s) = \max\!\left(0,\; \frac{\mathcal{W}_n^{(m)}(t, s)}{\theta_{n,s}^{(m)}} - 1\right)$$

A **K-of-scales gate** (minimum $k_s = 1$ scale must exceed threshold) suppresses isolated single-scale artefacts:

$$Z_n^{(m)}(t, s) \leftarrow 0 \quad \text{if} \quad \sum_s \mathbf{1}\!\left[\mathcal{W}_n^{(m)}(t,s) > \theta_{n,s}^{(m)}\right] < k_s$$

### 3.5 Multi-Moment Aggregation with Factorial-Inverse Weights

Moments are combined with weights inversely proportional to their factorial, emphasising stable lower moments:

$$w_m = \frac{1}{m!} \qquad \Rightarrow \qquad w_1 = 1,\; w_2 = \tfrac{1}{2},\; w_3 \approx 0.167,\; w_4 \approx 0.042$$

The signed aggregate energy across moments and scales is:

$$E_n(t) = \sum_{m} w_m \sum_s \left[ Z_n^{(m)}(t, s) - Z_n^{(m,-)}(t, s) \right]$$

where $Z^{(m)}$ corresponds to upward exceedance (positive deviation from baseline) and $Z^{(m,-)}$ to downward exceedance. The **signed aggregate across channels** is:

$$E_\text{signed}(t) = \sum_n E_n(t)$$

A Gaussian smoothing window of width 12 is applied to $E_\text{signed}$ before differentiation.

### 3.6 Derivative and Peak Extraction

The energy derivative detects the onset of changes:

$$\Delta E(t) = E_\text{signed}(t) - E_\text{signed}(t-1)$$

**Onset peaks**: local maxima of $\Delta E$ exceeding calibrated threshold $\varepsilon_\text{on}^+$ (positive energy step).
**Offset peaks**: local minima of $\Delta E$ below threshold $\varepsilon_\text{on}^-$ (negative energy step — variance decrease).

A refractory period of 150 samples prevents multiple detections near the same event.

### 3.7 Step Validation (Baseline Comparison)

For each candidate peak $\tau$, a level-shift test compares the post-peak region against the baseline:

$$\delta_\tau = \text{median}\!\left(E_\text{signed}[\tau + g : \tau + g + W_p]\right) - \text{median}\!\left(E_\text{signed}[\mathcal{B}]\right)$$

where gap $g = 10$, post-window $W_p = 100$. The baseline scale is estimated via MAD:

$$\hat\sigma_\mathcal{B} = 1.4826 \cdot \text{median}\!\left|E_\text{signed}[\mathcal{B}] - \text{median}(E_\text{signed}[\mathcal{B}])\right|$$

Peak $\tau$ is retained only if:

$$\delta_\tau > k_\delta \cdot \hat\sigma_\mathcal{B} \quad \text{(onset)}, \qquad \delta_\tau < -k_\delta \cdot \hat\sigma_\mathcal{B} \quad \text{(offset)}$$

with $k_\delta = 1.5$.

### 3.8 K-of-Channels Consistency Gate (v1-F)

This gate rejects peaks driven by a single channel. For each candidate peak $\tau$ and each channel $n$, the signed channel energy step is:

$$\Delta_n(\tau) = \text{median}\!\left(E_n[\tau + g : \tau + g + W_c]\right) - \text{median}\!\left(E_n[\tau - W_c - g : \tau - g]\right)$$

Normalised by a per-channel robust scale $\hat\sigma_n$ (estimated via MAD on the baseline):

$$z_n(\tau) = \frac{\Delta_n(\tau)}{\hat\sigma_n}$$

A channel is deemed **active** if $|z_n(\tau)| \geq k_z = 1.5$. Peak $\tau$ is retained only if:

$$N_\text{active}(\tau) = \sum_n \mathbf{1}\!\left[|z_n(\tau)| \geq k_z\right] \geq k_\text{ch}$$

with minimum $k_\text{ch} = 2$ channels required. For $p = 1$ this gate is automatically disabled.

A **concentration ratio** is also computed:

$$R(\tau) = \frac{\max_n |\Delta_n(\tau)|}{\sum_n |\Delta_n(\tau)|} \in \left[\tfrac{1}{p}, 1\right]$$

$R \approx 1$ indicates a single-channel artefact; $R \approx 1/p$ indicates a distributed change across all channels.

### 3.9 Near-Duplicate Merging

After all filtering, any two change points within 200 samples are merged into their midpoint:

$$\text{if } |\hat\tau_i - \hat\tau_j| < 200: \quad \hat\tau_\text{merged} = \left\lfloor \frac{\hat\tau_i + \hat\tau_j}{2} \right\rfloor$$

Applied greedily on the sorted list, left to right. An additional `simplify_transitions` pass with minimum distance 160 is applied in the evaluation stage.

## 4. Causal Structure Recovery

### 4.1 LiNGAM Identifiability

For each detected window $[t_r, t_{r+1})$, the data-generating process is assumed to be a linear non-Gaussian acyclic model (LiNGAM):

$$\mathbf{X} = \mathbf{B}^{(r)} \mathbf{X} + \boldsymbol{\varepsilon}, \qquad \varepsilon_i \overset{\text{indep}}{\sim} \mathcal{L}_i \text{ (non-Gaussian)}$$

The strict non-Gaussianity of each $\mathcal{L}_i$ guarantees identifiability of the unique acyclic $\mathbf{B}^{(r)}$ (Shimizu et al., 2006). DirectLiNGAM recovers the causal ordering by iteratively identifying the most exogenous variable via independence tests, then computing the weight matrix by ordinary least squares.

### 4.2 CausalMorph Warm-Start

The key challenge when applying LiNGAM across consecutive windows is that the causal order may drift, making cold re-fitting unreliable. CausalMorph addresses this by transforming the data under the prior structure before fitting:

Given prior causal order $\pi^{(r-1)}$ and prior adjacency $\hat{\mathbf{B}}^{(r-1)}$ (binarised at threshold 0.05), CausalMorph applies a residualisation to obtain a pre-whitened data matrix:

$$\mathbf{X}' = (\mathbf{I} - \hat{\mathbf{B}}^{(r-1)}) \mathbf{X}$$

DirectLiNGAM is then fitted on $\mathbf{X}'$, which is closer to the noise residuals and thus easier to order correctly when the structure has changed only partially.

The warm-start chain is:

$$(\pi^{(0)}, \hat{\mathbf{B}}^{(0)}) \;\xrightarrow{\text{CausalMorph}}\; (\pi^{(1)}, \hat{\mathbf{B}}^{(1)}) \;\xrightarrow{\text{CausalMorph}}\; \cdots$$

where $(\pi^{(0)}, \hat{\mathbf{B}}^{(0)})$ is bootstrapped from the known true structure of the first regime.

### 4.3 Structural Accuracy Metrics (Per Regime)

Let $\mathbf{A}^* \in \{0,1\}^{p \times p}$ be the true binary adjacency (diagonal zero) and $\hat{\mathbf{A}} = \mathbf{1}[|\hat{\mathbf{B}}| > 0.05]$ be the estimated binary adjacency. Define:

$$\text{TP} = \sum_{i \neq j} A^*_{ij}\,\hat{A}_{ij}, \quad
  \text{FP} = \sum_{i \neq j} (1-A^*_{ij})\,\hat{A}_{ij}, \quad
  \text{FN} = \sum_{i \neq j} A^*_{ij}(1-\hat{A}_{ij})$$

**Structural Hamming Distance:**

$$\text{SHD} = \text{FP} + \text{FN}$$

**Normalised SHD** (bounded in $[0,1]$, lower is better):

$$\overline{\text{SHD}} = \frac{\text{SHD}}{p(p-1)}$$

where $p(p-1)$ is the number of off-diagonal entries.

**Precision, Recall, F1:**

$$\text{Prec} = \frac{\text{TP}}{\text{TP}+\text{FP}}, \quad
  \text{Rec} = \frac{\text{TP}}{\text{TP}+\text{FN}}, \quad
  F_1 = \frac{2\cdot\text{Prec}\cdot\text{Rec}}{\text{Prec}+\text{Rec}}$$

## 5. Bayesian Structure Aggregation

### 5.1 Motivation

After recovering $R$ regime structures $\{\hat{\mathbf{A}}^{(r)}\}_{r=0}^{R-1}$, we collapse them into a single consensus graph. Because later regimes approximate the target structure $G_\text{tgt}$ better than early ones, we weight each regime's vote by its relative length.

### 5.2 Beta-Bernoulli Model

For each directed edge $(i, j)$ independently:

**Prior:** Edge presence $\theta_{ij} \sim \text{Beta}(\alpha_0, \beta_0)$ with $\alpha_0 = \beta_0 = 1$ (uniform / non-informative).

**Regime weights:** Let $n_r$ be the number of samples in regime $r$. The normalised weight is:

$$w_r = \frac{n_r}{\sum_{r'} n_{r'}}, \qquad \sum_r w_r = 1$$

**Weighted vote:** The length-weighted empirical edge frequency is:

$$\bar{x}_{ij} = \sum_{r=0}^{R-1} w_r \cdot \hat{A}^{(r)}_{ij} \;\in [0, 1]$$

**Effective count:** Scaling back to $n_\text{eff} = R$ regimes for concentration:

$$k_{ij} = R \cdot \bar{x}_{ij}, \qquad \bar{k}_{ij} = R - k_{ij}$$

**Posterior:** By conjugacy:

$$\theta_{ij} \mid \text{data} \;\sim\; \text{Beta}\!\left(\alpha_0 + k_{ij},\; \beta_0 + \bar{k}_{ij}\right)$$

**Posterior mean** (MAP for symmetric Beta priors):

$$\hat\theta_{ij} = \frac{\alpha_0 + k_{ij}}{\alpha_0 + \beta_0 + R} = \frac{1 + R\,\bar{x}_{ij}}{2 + R}$$

**Decision rule:** Include edge $(i,j)$ in the consensus if:

$$\hat\theta_{ij} > \theta_\text{cut} = 0.20$$

At threshold 0.20, an edge needs to appear (weighted) in roughly $\lfloor 0.20 \cdot (2+R) - 1 \rfloor / R \approx 20\%$ of regimes.

### 5.3 Properties

- **Shrinkage toward 0.5:** With $R$ small, the posterior is dominated by the prior and pulled toward 0.5 from both sides.
- **Monotone in data:** $\hat\theta_{ij}$ is monotonically increasing in $\bar{x}_{ij}$, so edges that appear in more/longer regimes always receive higher posterior probability.
- **Low threshold rationale:** Since early-trajectory regimes intentionally lack edges present only in $G_\text{tgt}$, a permissive threshold (0.20 vs. the classical 0.50) compensates for the trajectory-induced dilution.

## 6. Change-Point Detection Evaluation

### 6.1 Tolerance-Based Matching

A detected change point $\hat\tau$ is a **true positive** if $\exists\, \tau^* \in \mathcal{T}^*$ with $|\hat\tau - \tau^*| \leq \delta$ (tolerance $\delta = 125$ samples). Each ground-truth point is matched at most once.

$$\text{TP}_\text{det} = |\{(\hat\tau, \tau^*) : |\hat\tau - \tau^*| \leq \delta\}| \;\text{(one-to-one)}$$
$$\text{FP}_\text{det} = |\hat{\mathcal{T}}| - \text{TP}_\text{det}, \qquad \text{FN}_\text{det} = |\mathcal{T}^*| - \text{TP}_\text{det}$$

$$F_1^\text{det} = \frac{2\,\text{TP}_\text{det}}{2\,\text{TP}_\text{det} + \text{FP}_\text{det} + \text{FN}_\text{det}}$$

### 6.2 True Regime Assignment

For evaluation, each detected window $[t_r, t_{r+1})$ is matched to the ground-truth regime with the greatest temporal overlap:

$$r^*(r) = \arg\max_k \;\left|\left[t_r, t_{r+1}\right) \cap \left[\tau_{k-1}, \tau_k\right)\right|$$

The per-window SHD is then computed against $\mathbf{A}^{*\,(r^*(r))}$.

## 7. Parameter Summary

| Parameter | Value | Role |
|-----------|-------|------|
| $\rho_\text{base}$ | 0.35 | Base edge probability |
| $p$ | 3–7 | Number of nodes |
| $R$ | 5 | Number of regimes |
| $T_r$ | 600–800 | Samples per regime |
| $W$ | 50 | Rolling moment window |
| $s_\text{min}$ | 3.0 | Minimum CWT scale |
| $\omega_0$ | 6.0 | Morlet central frequency |
| $K$ | 100 | Surrogates |
| $\alpha$ | 0.40 | Surrogate significance level |
| $k_s$ | 1 | Min scales for K-of-scales gate |
| $k_z$ | 1.5 | Per-channel z-score threshold |
| $k_\text{ch}$ | 2 | Min active channels (v1-F gate) |
| $k_\delta$ | 1.5 | Step validation multiplier |
| $\delta_\text{merge}$ | 200 | Near-duplicate merging radius |
| $\delta$ | 125 | Detection tolerance |
| $\theta_\text{cut}$ | 0.20 | Consensus edge threshold |
| $\alpha_0, \beta_0$ | 1.0 | Beta prior hyperparameters |
